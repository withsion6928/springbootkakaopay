## 개발 환경 및 실행방법
- 기본 환경
    - IDE: IntelliJ IDEA Ultimate
    - OS: Windows10
    - GIT

- Server
    - jdk-13.0.1
    - Spring Boot 2.4.5
    - JPA
    - H2
    - maven
    - Junit5


- 실행 방법
  - maven reload->Spring boot 기동(kakapay) 후
    아래 url 접속 후 각 기능별 try it out
    - http://localhost:8088/swagger-ui/index.html#/TestSubmit
  
  
- 문제해결 방법
  - JPA native Query를 이용한 쿼리 조회 방식으로 처리
